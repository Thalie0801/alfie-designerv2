# Règles de comptage & quotas
## Images
- +1 pour chaque image **IA** générée/retouchée (Nano‑Banana).
- 0 si usage de template Canva **sans** image IA.
## Carrousel
- +1 **Image** pour chaque illustration IA incluse ; slides purement typographiques = 0.
## Reels
- +1 **Reel** par **export** MP4.
- **0 Woof** si Image→Vidéo ou Éco T2V ; **Woofs** uniquement si Premium T2V.
## Alerte
- Alerte **80 %** sur Images/Reels/Woofs.
## Stockage
- Rétention **30 jours** par défaut ; add‑on 90 jours.
