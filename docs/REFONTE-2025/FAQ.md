# FAQ (V1)

## Modes d'interaction
**Chat Alfie vs Créateur ?** → **Chat Alfie** (`/chat`) pour explorer et être guidé, **Créateur** (`/app`) pour générer rapidement avec contrôle total. Le Chat pose des questions, le Créateur affiche un formulaire direct.  
**Quand utiliser quel mode ?** → Débutants/idéation → Chat. Production quotidienne/workflows → Créateur.

## Livraison & Publication
**Push vs Pull ?** → V1 ne fait **que du PULL** : on fournit un lien "Ouvrir dans Canva" + un ZIP prêt à poster.  
**Publication automatique ?** → Non. Designs prêts à planifier.  
**Quand utiliser Premium (Woofs) ?** → Pour des plans "héros" réalistes/ciné. Sinon préférer Éco T2V ou Image→Vidéo.

## Comptage rapide
- Image template **sans IA** : 0 image comptabilisée.
- Carrousel : images IA intégrées → décompte dans **Images**.
- Reel : **chaque export** = 1 Reel. **0 Woof** pour Image→Vidéo et Éco T2V.
