# Checklists Qualité & Accessibilité
- Contraste **AA** vérifié (texte sur fond).
- Hiérarchie typographique (H1/H2/captions) et marges de sécurité respectées.
- **Alt‑texts** fournis (format “Slide X — …”).
- Vidéo : hook initial 0–2s ; 5–7 beats ; sous‑titres ≤2 lignes ; **−14 LUFS** ; safe‑zones 9:16 ; pas de flash excessif.
- Fichiers livrés : MP4 + SRT + cover (vidéo) ; PNG/SVG/PDF selon format.
