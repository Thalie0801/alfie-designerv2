# Critères d’acceptation

## Commun

- [ ] Invite unique → questions (2–3) → routage → prévisualisation → livraison PULL
- [ ] Bouton **Ouvrir dans Canva** (lien fonctionnel)
- [ ] **ZIP** téléchargeable (nommage standard: `brand-type-YYYYMMDD-hhmm.zip`)
- [ ] Compteurs visibles (images / reels / woofs) + **alerte 80%**
- [ ] Pas de publication automatique

## Image

- [ ] Export PNG + (option PDF)
- [ ] Comptage conforme (IA=+1, template sans IA=0)
- [ ] Alt-text proposé

## Carrousel (IG/LinkedIn)

- [ ] 5–10 slides (configurable)
- [ ] Slides IA comptées dans **Images**, slides texte-only=0
- [ ] Export PNG (bundle)

## Reel 9:16 (8–15s)

- [ ] Hook 0–2s, 5–7 beats, **−14 LUFS**
- [ ] Sous-titres **≤ 2 lignes** (SRT)
- [ ] Export MP4 + SRT + cover
- [ ] **Woof** consommé **uniquement** si **Premium T2V** (confirmation préalable)

## Accessibilité & qualité

- [ ] Contraste **AA**, hiérarchie H1/H2/H3
- [ ] Safe-zones 9:16 respectées
